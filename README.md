## 📌 Featured Project (Internship)

### 🚀 Finite Air: AI-Powered Custom Design Factory
I built this SaaS prototype from scratch, focusing on backend architecture and API integrations. I took full ownership of the product's visual identity, **designing the GUI, selecting the optimal frontend tools, and building the interface from the ground up.**

https://github.com/user-attachments/assets/33016dee-3209-413c-8120-ceee430d12b1

**Key Deliverables:**
- **🎨 UI/UX & Frontend:** Designed the GUI and built a responsive interface from scratch using **TailwindCSS**.
- **🔐 Secure Infrastructure:** Implemented user authentication with mandatory email activation flows.
- **🤖 AI Integration & Webhooks:** Integrated an AI Designer via API and configured **Webhooks** for real-time communication and persistent chat history.
- **🧩 3D Asset Management:** Developed a "garage" with **GLB previews**, search functionality, and custom filtering for 3D models.
- **💳 Payment Systems:** Integrated a full **Stripe** subscription flow, managing automated billing and payment validation.
- **⚙️ DevOps & Stability:** Dockerized the entire application, managed PostgreSQL migrations, and implemented system health checks.
- **✅ Reliability:** Developed automated tests for critical user and transaction flows.

**Stack:** Python (Django), PostgreSQL, Docker, TailwindCSS, Stripe API, REST APIs & Webhooks.
